# 用户故事：Brianne 想让全家一起把电用对

> 改写自 [Brianne Myers · Demo 反馈 · 2026-02-05](../../interviews/s3-demo-2.0/2026-02-05-Brianne%20Myers-demo.md)。
> ⚠️ **低信息量场次**：Brianne 全程**未真正操作 App**（链接没收到，只听了讲解），所有"喜欢"都是对口头描述的反应；她**是否自有太阳能/EV/电池均未确认**。本故事忠实呈现她说过的话，但严格说是"她期待中的样子"，仍需可交互 Demo 上手后复访验证。

## 画像速览
- **谁**：Brianne Myers（Zoom 显示名 "Thee Citizen"），和家人同住。
- **痛点**：家里用电难管理，**希望全家人都能更聪明地用电、一起把账单压下来**。
- **关系**：不是第一次接触 Casally——她想要的东西"正是第一次会议时就想要的"；目前是 Casally 的有偿早期测试用户。
- **设备 / 住房 / 地区**：均未说明（不确认其自有太阳能等）。

## 故事（约 100 字）
> Brianne 和家人同住，一直头疼怎么让全家更聪明地用电。听完 demo，她最喜欢"节省"功能——能看到一个月到底省了多少钱，还能追踪峰谷时段、知道什么时候该用太阳能、什么时候别用电，所有细节 App 里都看得到。她说：这正是我想要的。听到还能添加家庭成员，她直呼 so awesome。

## 她真正认可的价值点（唯一可沿用的正向锚点）
- **"看省了多少钱"（Saving）**：她主动说至少喜欢这部分，因为"能知道这个月省了多少"，并和"第一次会议的诉求"对上。
- **看峰谷时段决定何时用电**：知道 off-peak 什么时候能用、什么时候别用。
- **添加家庭成员**：直接命中她"想让全家一起更聪明用电"的痛点。

## 复访时该补问的背景（数据缺口）
- 住房/地区、家庭人数与用电场景；是否自有太阳能/EV/电池/智能面板。
- "太阳能充满就不能用电"那句到底指自家真实情况还是顺演示在说。
- 可交互 Demo 上手后，用她**真实账单 + 用电习惯**重做一次实操访谈，才能拿到有效反馈。
