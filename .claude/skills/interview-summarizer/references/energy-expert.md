# 提炼维度：专业咨询（energy-expert）

骨架而非死模板：维度是抓手不是死清单，原文展开到哪就挖到哪，专家的关键判断、反直觉观点要原汁原味记清、宁细勿漏。

## 必提维度
- **专业观点 / 行业知识点**：专家给出的事实、判断、经验。
- **对产品方向的专业建议**：建议做什么 / 别做什么及理由。
- **技术 / 政策 / 术语**：电价结构、净计量(NEM)、储能、TOU、关税补贴等，简要解释。
- **合规 / 认证路径**（若涉及）：FCC、NRTL、UL、安规、DOE 能效、Prop 65 等的要求与流程。
- **监管 / 门槛与成本**：认证 / 合规的费用、周期、地区差异、强制与否、谁来买单。
- **风险 / 坑 / 现实障碍**：专家点名的常见错误、容易低估的难点、落地拦路点。

## 注意
- 区分"已被验证的事实"与"专家的个人推测"（后者按《提炼原则》标注）。
- 保留关键数字、单位、政策名称、机构名的准确原文。

> 深度与忠实并重，见 SKILL.md《提炼原则：又深又准》。
